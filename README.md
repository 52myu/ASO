# ASO

>> 此项目主要为，IOS端AppStore进行刷榜操作

  ----
  
  
# 项目结构

|文件名|描述|功能|
|---|-----|------|
|ASO|AppStore Search optimize 的相关逻辑代码|包含注入点 ，改吗，过验证码，卸载，模拟点击 等|
|layout|一些常用的图标文件|用于美化插件|
|lib|插件需要的一些依赖，我已经将其打成支持各个手机arm的lib||
|Makefile|项目的编译管理||
|Tweak.xm|注入点的声明||
|NGASO.plist|当前此项目的注入程序的进程||
